# DeepWhite 分镜提示词构建器（MM-KR适配版）

来源：deepwhite-shotlist-builder-zh-user v2.7。

## 适用位置

Phase 5 · 版本C，与情绪导演（A）和HuggingField（B）并列输出。

## 提示词固定结构顺序

1. **固定开头**：`不要出现BGM，不要出现字幕`
2. **风格四段**：`【全局画质】`、`【人物材质】`、`【灯光与风格】`、`【核心特效】`
3. **素材句柄声明**：`@image1`、`@image2` 等
4. **空间与镜头数量警告**
5. **镜头段落**：`【镜头N｜起止秒】`
6. **环境活动/全场音效**
7. **防错警告**

## 风格四段（按参考图画风适配）

```
【全局画质】{profile.art_style}画风，高品质手绘质感，非实拍、非三维渲染、非游戏引擎；画面具备画风特有的笔触层次和质感。
【人物材质】皮肤细腻光泽和色温变化，发丝有笔触感的光泽层次，眼眶和嘴唇的色彩饱和度与周围肤色形成微妙差异，高光处有晕染。
【灯光与风格】{profile.aesthetic_keywords}；光源位置和色温按场景设定。
【核心特效】[按场景填写：室内灯光反射、窗外城市灯光散射、雨水/酒杯水珠等物理视觉机制]
```

## 素材句柄声明规则

每条提示词重新编号句柄：

```
@image1（角色名）——[逐项对照人设图的外貌/服装/面部/饰品]。当前状态：[本场景的具体状态]。
@image2（VIP室）——[场景布局描述，多视图说明上/下/中各拍了什么]。
```

- 人物句柄：身高、体型、发型、面部识别点、从头到脚服装、随身道具
- 场景句柄：参考图布局说明
- 服装描述必须逐项对照人设图/分镜图

## 镜头段落字段顺序

每个镜头的固定字段顺序：

```
【镜头N｜X.X-Y.Y秒】
画面动作概述：一句话概括主要动作和人物状态。
画面构图：初始画面构图，使用屏幕区域（左三分之一/中央/右三分之一/前景/中景/远景）和空间关系（前方/后方/左侧/右侧/遮挡/接触点）。
机位：摄影机位置、景别、焦段、运镜方式。镜头自然晃动模拟手持拍摄。
动作：按时间顺序的人物动作、物体运动、微表演。
音效：本镜头声音。
```

## 摄影机与情绪同步

| 角色情绪 | 摄影机类型 |
|---|---|
| 愤怒/紧张/边缘 | 不安手持，呼吸式漂移，微抖 |
| 冷静/掌控/自信 | 克制手持，极细微规律呼吸感 |
| 悲伤/脆弱 | 低位缓慢手持，呼吸节奏放慢 |
| 震惊/揭示 | 开始静止，极慢推进或拉远 |

## 表演微节拍（禁止泛情绪标签）

情绪必须拆为肌肉、呼吸、眼神、皮肤、姿态：

### 愤怒/决心
- 咬肌透过皮肤收紧、鼻翼在重音词扩张、瞳孔收缩视线变窄、太阳穴细汗珠

### 焦虑/紧张
- 喉结吞咽、台词前短促鼻吸气、舌尖舔润下唇、脸颊毛细血管泛红

### 悲伤/脆弱（不落泪版）
- 眼角外缘下垂、眼眶湿润不落泪、眉头中央皱起、嘴唇轻微颤抖一次

### 控制/冷静/优越
- 呼吸平稳、手指放松、缓慢均匀眨眼、下颌微抬、微笑逐渐建立

### 压抑情绪
- 颧骨咬肌慢慢收缩、延迟用力吞咽、眼眶水光不落泪、深控制吸气胸腔起伏

## 台词前后微表演

```
台词前：一次短促鼻吸气，喉结吞咽。
台词："[韩语台词]"——重音落在"[字]"，[鼻翼/瞳孔/下颌反应]。
台词后：视线继续锁定[方向]0.5秒，然后微微向下移开。
```

## 时长规则

- 单条提示词最多15秒
- 时长 = 韩语台词音节数÷6~8 + 动作缓冲
- 目标8-12秒
- 多镜头内部时长之和 = 提示词总时长

## 字数控制

- 每条可复制提示词正文 ≤2200中文字符
- 目标范围1800-2200字
- 超限时压缩顺序：删重复风格词 → 合并环境描述 → 缩短非关键音效 → 减少普通防错 → 保留核心

## 防错警告

- `⚠️` 普通警告，`⚠️⚠️⚠️` 关键警告
- 常见：身份漂移、句柄污染、道具错位、多加镜头、焦点漂移、空间镜像、背景空掉

## 多镜头 vs 单镜头

多镜头开头：`多机位剪辑。`
单镜头开头：`单镜头，无剪辑。预计时长约X秒。`

## MM固定规则叠加

在DeepWhite分镜提示词之上，仍需遵守MM固定规则：
- 镜头自然晃动模拟手持拍摄
- [图1]人设图 [图2]分镜图 引用格式（与@image1/@image2对应）
- 禁止引用拟声词音节
- 服装描述逐项对照人设图/分镜图
- `{profile.language}` 台词保持原文不翻译
