# Seedance提示词 · 版本A · 情绪导演六段式

总字数 ≤1900字（含标点）

## 模板

```
一、视听限制
无背景音乐，纯环境音。[具体环境音列表]。画面无字幕无水印。全程参考[图1]角色形象与[图2]构图。

二、语言台词
全程{profile.language}，[性别]声。[语气变化描述]。
台词：（[音量/音质标注]）"[{profile.language}台词]"

三、运镜手法
[运镜方式]，镜头自然晃动模拟手持拍摄。[运动轨迹]。

四、风格色调与光景
{profile.art_style}。{profile.aesthetic_keywords}[冷暖倾向]。[光源位置与色温]。

五、角色与场景设定
角色：参考[图1]。[逐项外貌：发型发色/面部/服装/身材]。
场景：[时间][地点]，[环境要素]。

六、时间轴详细叙事（建议时长：Xs）
【0-Xs：段落标题】
景别：[景别]
动作：[起点→过程→终点，用身体部位描述]
神态：[面部肌肉具体运动，禁止情绪标签]
台词：（[音量/音质]）"[{profile.language}]"

【Xs-Ys：段落标题】
...
```

## 核心规则
- 禁止比喻/拟人/象征/夸张等修辞
- 情绪用生理表现描述（肌肉运动/皮肤状态/呼吸变化）
- 距离用身体参照（一臂距离/人物占画面比例）
- 禁止精确物理数值（厘米/角度/赫兹）
- 台词标注：语速/音量/音质/情绪声音
- 时间轴按关键节点分段

## 字数超限处理优先级
1. 压缩时间轴重复动作
2. 简化环境音列举
3. 合并次要外貌细节
4. 缩减台词长度
5. 按关键节点分段替代逐秒拆分
