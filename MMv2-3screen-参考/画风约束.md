# 画风与技术约束

## 画风
以用户提供的参考图为准。通过 image edit API 以参考图为输入，自动延续参考图的画风。

如果用户没有参考图，使用 `{profile.art_style}` + `{profile.aesthetic_keywords}` 作为画风参数。

## 图片生成约束
- 模型：gpt-image-2
- 接口：`/v1/images/edits`（image edit，以参考图为输入）
- 禁止使用 `/v1/images/generations`（纯文本生成，角色不一致）
- 人设图分辨率：4K（1536x1536 或 API支持的最大正方形）
- 分镜图分辨率：4K（宫格排列，每帧9:16比例）
- 质量：high
- 格式：png

## 角色表现约束
- 服装描述逐项对照人设图/分镜图原图，禁止凭记忆
- 禁止夸张瞪眼、嘴巴大张等AI味浓的表情
- 表情自然克制，参考参考图画风对应的表演风格
- 肢体动作不要过于僵硬或对称

## 图片内容约束
- 禁止任何文字、水印、标签
- 暴露度适中：不过度暴露也不过度遮盖
